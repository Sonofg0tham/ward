# Ward Brand Identity

Direction: cut-iron sentinel - calm, engraved, precise. A gatekeeper, not an alarm.

Fonts:
- Display: Clash Display (via Fontshare)
- Body: General Sans (via Fontshare)
- Mono: IBM Plex Mono (via Google Fonts)

Palette (HSL):
- Background: hsl(20 8% 8%)        /* iron-tinted warm dark, never #000 */
- Foreground: hsl(30 12% 91%)      /* off-white, never pure #FFF */
- Accent: hsl(20 78% 52%)          /* warded ember - rust-orange, used sparingly */
- Border: hsl(20 8% 20%)
- Muted: hsl(20 6% 62%)

Light-mode counterparts (for the light logo variant and light surfaces):
- Background: hsl(30 20% 97%)
- Foreground: hsl(20 10% 12%)
- Border: hsl(25 12% 86%)

Radius scale: 0.375rem (tools are precise; keep corners tight)

Signature detail: the ward cut - a single 45-degree stencil notch cut
through the strokes of the W monogram. The same cut angle recurs as the
motif across the brand: badge corners, section dividers, list markers.

Marketplace mapping: GitHub Action branding is locked to Feather icons +
8 colours; Ward uses icon "shield", color "orange" (closest to warded
ember).

Reserved for this product. Not sonofg0tham. Not any other product Craig
has built.
